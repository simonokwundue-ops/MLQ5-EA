# MLQ5-EA
Creating Expert Advisors for MetaTrader 5

## Overview

This repository contains a professional MQL5 Expert Advisor (EA) developed based on comprehensive forex trading educational materials. The EA implements proven trading strategies with professional risk and money management.

## Contents

- **ForexMasterEA.mq5** - Fully functional MQL5 Expert Advisor
- **README_EA.md** - Comprehensive documentation for the EA
- **Forex Educational Materials** (PDF documents):
  - Ebook_Forex-A-Z_copy.pdf
  - Forex-Trading-For-Beginners-The-Ultimate-Guide_copy.pdf
  - Trading_Course_Advanced.pdf

## Quick Start

1. Copy `ForexMasterEA.mq5` to your MetaTrader 5 `MQL5/Experts` folder
2. Compile the EA in MetaEditor
3. Attach to a chart and configure parameters
4. Enable AutoTrading

For detailed instructions, see [README_EA.md](README_EA.md)

## Features

- Moving Average crossover strategy
- ADX trend strength confirmation
- Professional risk management (2% per trade default)
- Dynamic position sizing
- Stop loss and take profit management
- Spread and trading hours filters
